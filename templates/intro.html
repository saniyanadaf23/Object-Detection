How It Works

A simple 3-step flow to run YOLO11 efficiently for your experiments and project demos.

1. Login / Sign up

Create a basic account, log in, and access the detection dashboard built with Flask and Bootstrap.

2. Choose Mode

Select between image, video, or webcam mode depending on what you want to analyze in real-time.

3. Run YOLO11

The backend model performs detection and streams back visual results with bounding boxes and labels.
